Overview
The Smart Water Advisor is an AI-powered conversational system designed to promote responsible water usage and conservation. The project is aligned with the United Nations Sustainable Development Goal 6 (Clean Water and Sanitation).

Problem Statement
Water scarcity and inefficient water usage are major sustainability challenges. There is a need for accessible, reliable guidance to help individuals adopt water-saving practices.

AI Solution
This project uses IBM watsonx Prompt Lab to build a structured, rule-based generative AI system that provides accurate and responsible water conservation advice.

Technologies Used
- IBM watsonx Prompt Lab
- Granite / LLaMA Instruction Models
- Structured Prompt Engineering
- Responsible AI Guardrails

 SDG Alignment
- SDG 6: Clean Water and Sanitation

 Testing
The deployed AI prompt was tested using real-world water sustainability queries in the Prompt Lab interface. Responses were evaluated for relevance, clarity, and ethical alignment.
